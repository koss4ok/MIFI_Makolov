<h2>Генерация армии противника</h2>
<h4>Алгоритм: Жадный</h4>
<h4>Сложность: O(n * log(n))</h4>
<br>
<h2>Симуляция боя</h2>
<h4>Алгоритм: Жадный</h4>
<h4>Сложность: O(r * n log(n)), где r - количество раундов.</h4>
<br>
<h2>Отбор целей для атаки</h2>
<h4>Алгоритм: hash-based</h4>
<h4>Сложность: O(n * m)</h4>
<br>
<h2>Поиск кратчайшего пути</h2>
<h4>Алгоритм: BFS</h4>
<h4>Сложность: O(w * h)</h4>
